# Movie Revenue Prediction using Machine Learning

## Project Overview
This project predicts movie revenue using Machine Learning techniques and movie metadata.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Features
- Data preprocessing
- Dataset merging
- Feature engineering
- Random Forest Regression Model
- Performance Evaluation using MAE, RMSE, and R² Score
- Data Visualization

## Dataset
TMDB 5000 Movie Dataset

## Workflow
1. Data Collection
2. Data Cleaning
3. Dataset Merging
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Prediction
8. Evaluation
9. Visualization

## Results
Successfully developed a Random Forest Regression model to predict movie revenue using movie metadata and visualization techniques.
